Detailed Description

Parameters for three body angular potential (bond-angle potentials).

ATTENTION: Note that there are different implementations of the bond angle potential which you may chose with a compiler flag in the file config.hpp ! bend - bending constant. phi0 - equilibrium angle (default is 180 degrees / Pi)
