Detailed Description

All 3d non P3M methods to deal with the magnetic dipoles.

DAWAANR => DIPOLAR_ALL_WITH_ALL_AND_NO_REPLICA Handling of a system of dipoles where no replicas Assumes minimum image convention for those axis in which the system is periodic as defined by setmd.

MDDS => Calculates dipole-dipole interaction of a perioidic system by explicitly summing the dipole-dipole interaction over several copies of the system Uses spherical summation order
