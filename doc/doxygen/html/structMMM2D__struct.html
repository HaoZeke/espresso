int MMM2D_struct::far_calculated

if nonzero, MMM2D_struct::far_cut has been calculated by MMM2D_tune_far, and will be recalculated automatically, if important parameters, such as the number of cells, change.

If this is zero, the far cutoff has been set explicitly by the user and will not be touched by Espresso.
