157 /** numerical solution for the integration constant \f$\gamma\f$ in the cell model, determined by
158  \f[\gamma\,\ln\frac{R}{r_0}=\arctan\frac{1}{\gamma}+\arctan\frac{\xi_M-1}{\gamma}\f]
159  from which the second integration constant, the Manning radius \f$R_M\f$, follows to
160  \f[R_M = R\cdot\exp\left(-\frac{1}{\gamma}\cdot\arctan\frac{1}{\gamma}\right)\f]
161  Any value \f$\xi_M\geq 0\f$ is allowed, the function will automatically ensure the
162  analytical continuation required for \f$\xi_M<\ln(R/r_0)/(1+\ln(R/r_0))\f$, in which case
163  \f$\gamma\f$ becomes imaginary.
164  @param xi_m Manning parameter \f$\xi_M=\ell_B/a\f$ (with Bjerrum-length \f$\ell_B\f$ and charge distance \f$a\f$)
165  @param Rc outer radius \f$R_C\f$ of the cylindrical cell around each polyelectrolyte
166  @param ro inner radius \f$r_0\f$ of the cylindrical cell around each polyelectrolyte
167  @param gacc the accuracy up to which \f$\gamma\f$ should be determined
168  @param maxtry maximum number of interations to find a solution
169  @param result pointer to double array containing \f$\gamma\f$ and \f$R_M\f$,
170  and a third entry which is -1.0 if \f$\gamma\f$ is imaginary, +1.0 else. */

283 /** Calculates the van Hove auto correlation function and as a side product the mean sqaure displacement (msd).
284 
285  Calculates the van Hove auto correlation function (acf) G(r,t) which is the probability that a particle has moved
286  a distance r after time t. In the case of a random walk G(r,t)/(4 pi r*r) is a gaussian. The mean square
287  displacement (msd) is connected to the van Hove acf via sqrt(msd(t)) = int G(r,t) dr. This is very useful for
288  the investigation of diffusion processes.
289  calc_vanhove does the calculation for one particle type ptype and stores the functions specified by rmin, rmax and
290  rbins in the arrays msd and vanhove.
291 
292  @param ptype particle type for which the analysis should be performed
293  @param rmin minimal distance for G(r,t)

295  @param rbins number of bins for the r distribution in G(r,t)
296  @param tmax max time, for which G(r,t) is computed, if omitted or set to zero, default tmax=n_configs-1 is used
297  @param msd array to store the mean square displacement (size n_configs-1)
298  @param vanhove array to store G(r,t) (size (n_configs-1)*(rbins))

304 /** Calculates the spherically averaged structure factor.
305 
306  Calculates the spherically averaged structure factor of particles of a
307  given type. The possible wave vectors are given by q = 2PI/L sqrt(nx^2 + ny^2 + nz^2).
308  The S(q) is calculated up to a given length measured in 2PI/L (the recommended order of
309  the wave vector is less than 20).
310  The data is stored starting with q=1, and contains alternatingly S(q-1) and the number
311  of wave vectors l with l^2=q. Only if the second number is nonzero, the first is meaningful.
312  This means the q=1 entries are sf[0]=S(1) and sf[1]=1. For q=7, there are no possible wave vectors,
313  so sf[2*(7-1)]=sf[2*(7-1)+1]=0.
314 
315  @param type the type of the particles to be analyzed
316  @param order the maximum wave vector length in 2PI/L
317  @param sf pointer to hold the base of the array containing the result (size: 2*order^2).
318 */
