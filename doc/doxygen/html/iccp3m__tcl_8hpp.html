Implementation of the tcl-command
iccp3m { <last_ind_id> <e1> <num_iteration> <convergence> <relaxation> <area> <normal_components> <e_in/e_out> [<ext_field>] | iterate } ICC sets up and calculates induced charges on dielectric surfaces.

At the beginning of every simulation run particles on the surface boundary have to be set up (before any real particle) together with the list of areas, normal vectors and dielectric constant associated with them. After that the iterate flag can be used during the simulation to update the value of the induced charges.

Parameters:
<last_ind_id> ID of the last surface charge.
