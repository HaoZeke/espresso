void analyze_cwvac ( int  maxtau,
int  interval,
double **  _avac,
double **  _evac 
)

Calculates the (charge weighted) velocity auto-correlation function from the stored configurations.

The charge weighted velocity auto-correlation function is used to determine the electrophoretic mobility of a chain using Green-Kubo relation.

cwvac(tau) = < sum_i^N ( q_i*v_i(t0)*v_CM(t0+tau) ) >

Parameters
maxtaumaximal tau
intervalstep between t0, sampling frequency
_avaccontains the averaged(=over all chains in the system) velocity auto-correlation
_evaccontains the error associated with the averaged velocity auto-correlation function

_ffcontains S(q) as an array of size qbins
