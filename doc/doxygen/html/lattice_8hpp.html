Lattice class definition Contains the lattice layout and pointers to the data fields.

For parallelization purposes, it is assumed that a halo region surrounds the local lattice sites.

int lattice_switch

Switch determining the type of lattice dynamics.

A value of zero means that there is no lattice dynamics. Different types can be combined by or'ing the respective flags. So far, only LATTICE_OFF and LATTICE_LB exist.
