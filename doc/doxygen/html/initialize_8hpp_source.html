23 /** \file initialize.hpp This file contains the hook procedures. These
24  are the ones with names on_* and are called whenever something is
25  changed in Espresso which might influence other parts. For
26  example, the P3M code has to be reinitialized whenever the box
27  size changes. The hooking mechanism allows to keep track of such
28  changes.
29 
30  For this mechanism to work, two things have to be fulfilled. If
31  some part of the code changes some property, it has to call the
32  corresponding hook, i. e. on_particle_change if a particle
33  property has been changed or on_short_range_ia_change, if a short
34  ranged interaction has been changed. In turn procedures that
35  depend on particle properties or the box size, should react to
36  such changes in the corresponding hook procedure.
37  */
38 
39 /** \name Hook procedures
40  These procedures are called if several significant changes to
41  the system happen which may make a reinitialization of subsystems
42  necessary. Note that all these functions are called on ALL nodes.
43  if you need to do something only on the master node, check
44  \ref this_node == 0. The use of the asynchronous mpi_* functions
45  (e. g. mpi_bcast_parameter) on the master node is possible.
46 */
47 /*@{*/
