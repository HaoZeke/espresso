21 /** \file specfunc.hpp
22  This file contains implementations for some special functions which are needed by the MMM family of
23  algorithms. This are the modified Hurwitz zeta function and the modified Bessel functions of first
24  and second kind. The implementations are based on the GSL code (see \ref specfunc.cpp "specfunc.c"
25  for the original GSL header).
26 
27  The Hurwitz zeta function is evaluated using the Euler-MacLaurin summation formula, the Bessel functions
28  are evaluated using several different Chebychev expansions. Both achieve a precision of nearly machine
29  precision, which is no problem for the Hurwitz zeta function, which is only used when determining the
30  coefficients for the modified polygamma functions (see \ref mmm-common.hpp "mmm-common.h"). However, the
31  Bessel functions are actually used in the near formula of MMM2D, which is therefore slightly slower than
32  necessary. On the other hand, the number of terms in the Bessel sum is quite small normally, so that a less
33  precise version will probably not generate a huge computational speed improvement.
34 */
