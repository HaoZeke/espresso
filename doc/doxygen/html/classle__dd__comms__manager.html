Setup the comms order.

Allocate and fill a short int array with the order of comms.

Update a y-direction comm after a Lees-Edwards offset change.

Sending or receiving cells in Y direction with LE wrap means looking at the existing cell list with an offset.

This function fills the comm list using cells with an offset or no cells at all, depending on what is required.
