Detailed Description

Molecular dynamics integrator for rotational motion.

A velocity Verlet algorithm using quaternions is implemented to tackle rotational motion. A random torque and a friction term are added to provide the constant NVT conditions. Due to this feature all particles are treated as 3D objects with 3 translational and 3 rotational degrees of freedom if ROTATION flag is set in config.h.
