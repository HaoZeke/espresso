21 /** \file mmm2d.hpp MMM2D algorithm for long range coulomb interaction
22  in 2d+h geometries. Implementation of the MMM2D method for the
23  calculation of the electrostatic interaction for two dimensionally
24  periodic systems. For details on the method see MMM general. The
25  MMM2D method works only with the layered or nsquared \ref
26  tclcommand_cellsystem "cell system". The tuning is not automated,
27  since the only tunable parameter is the cell size, which can be
28  changed easily in Tcl. Moreover, only a few values make sense to
29  be tested, since in general the number of cells will be between 5
30  and 20.
31  */

61 /** set parameters for MMM2D. This assumes that the particles do NOT leave the box.
62  For the near formula (nsquared cell structure), precision might be lost, while
63  the far formula might have problems with overflows.
64  @param maxPWerror the maximal error for the pairwise interactions. Both for
65  potential and force components. The potential is therefore
66  always slightly more precise
67  @param far_cut sets the cutoff for the far formula in inverse lengths.
68  If -1, the far cutoff is determined by maxPWerror.
69  Manual setting is probably only good for testing
70  @param delta_top dielectric contrast at top of the simulation box
71  @param delta_mid dielectric contrast in the middle of the simulation box
72 */
73 int MMM2D_set_params(double maxPWerror, double far_cut, double delta_top, double delta_bot, int const_pot_on, double pot_diff);
