34 /** Tune P3M parameters to desired accuracy.
35 
36  Usage:
37  \verbatim inter coulomb <bjerrum> p3m tune accuracy <value> [r_cut <value> mesh <value> cao <value>] \endverbatim
38 
39  The parameters are tuned to obtain the desired accuracy in best
40  time, by running mpi_integrate(0) for several parameter sets.
41 
42  The function utilizes the analytic expression of the error estimate
43  for the P3M method in the book of Hockney and Eastwood (Eqn. 8.23) in
44  order to obtain the rms error in the force for a system of N randomly
45  distributed particles in a cubic box.
46  For the real space error the estimate of Kolafa/Perram is used.
47 
48  Parameter range if not given explicit values: For \ref p3m_parameter_struct::r_cut_iL
49  the function uses the values (\ref min_local_box_l -\ref #skin) /
50  (n * \ref box_l), n being an integer (this implies the assumption that \ref
51  p3m_parameter_struct::r_cut_iL is the largest cutoff in the system!). For \ref p3m_parameter_struct::mesh
52  the function uses the two values which matches best the
53  equation: number of mesh point = number of charged particles. For
54  \ref p3m_parameter_struct::cao the function considers all possible values.
55 
56  For each setting \ref p3m_parameter_struct::alpha_L is calculated assuming that the
57  error contributions of real and reciprocal space should be equal.
58 
59  After checking if the total error fulfils the accuracy goal the
60  time needed for one force calculation (including verlet list
61  update) is measured via \ref mpi_integrate (0).
62 
63  The function returns a log of the performed tuning.
64 
65  The function is based on routines of the program HE_Q.cpp written by M. Deserno.
66  */
67 int tclcommand_inter_coulomb_p3m_print_tune_parameters(Tcl_Interp *interp);
