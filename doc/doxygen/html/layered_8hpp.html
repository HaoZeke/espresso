Detailed Description

The layered cellsystem.

This cellsystem is a combination of a single processor n-squared method along x and y, and a multiprocessor domain decomposition along z. Therefore only processors grids are allowed for this cellsystem. The implementation is pretty similar to domain_decomposition.h.
