int double_correlation_get_data ( double_correlation *  self)

The function to process a new datapoint of A and B.

First the function finds out if it necessary to make some space for the new entries of A and B. Then, if necessary, it compresses old Values of A and B to make for the new value. Finally The new values of A and B are stored in A[newest[0]] and B[newest[0]], where the newest indices have been increased before. Finally the correlation estimate is updated. The initialization procedure for the correlation object.

All important parameters have to be speciefied at the same time. They can not be change later, so every instance of the correlation class has to be fed with correct data from the very beginning.

Parameters
selfThe pointer to the correlation class instance that should be initialized
dtThe time interval between subsequent updates
tau_linThe linear part of the correlation function.
tau_maxmaximal time delay tau to sample
window_distanceThe distance in time domain between update of the correlation estimate
dim_AThe dimension of the A vector
dim_BThe dimension of the B vector
AFirst observable to correlate
BSecond observable to correlate
dim_corrThe dimension of the correlation to be calculated
corr_operation_namehow to correlate the two observables A and B (this has no default)
compressA_namehow the A values should be compressed (usually the linear compression method)
compressB_namehow the B values should be compressed (usually the linear compression method)
argsthe parameters of the observables
