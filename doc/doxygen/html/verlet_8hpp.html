Detailed Description

This file contains routines to setup and handle interaction pair lists (verlet pair lists) for the non bonded interactions.

For the non-bonded interactions, the domain decomposition force calculation uses verlet pair lists which contain all particle pairs with a distance smaller than their maximal interaction range, based on their types, plus the skin. This allows one to use these verlet pair lists for several time steps, as long no particle has moved further than skin / 2.0. You can tune the verlet pair algorithm with the variable skin which you can set via the tclcommand_setmd command. You can also acces the average number of integration steps the verlet lists have been reused with tclcommand_setmd verlet_reuse.

The verlet algorithm uses the data type PairList to store interacting particle pairs.

To use verlet pair lists for the force calculation you can either use the functions build_verlet_lists and calculate_verlet_ia or a combination of those two build_verlet_lists_and_calc_verlet_ia.

For energy and pressure calculations using verlet pair lists use calculate_verlet_energies and calculate_verlet_virials.

For more information see verlet.c.
