Detailed Description

This file contains routines to perform metadynamics.

Right now, the reaction coordinate is defined between two particles (either distance or z-projected distance). Note that these particles can be virtual sites, in order to handle molecules.

  • set metadynamics options
  • initialize bias forces and free energy profiles
  • calculate reaction coordinate for each integration step

  • apply bias force on particles
