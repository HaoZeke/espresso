21 /** \file lb-boundaries.hpp
22  *
23  * Boundary conditions for Lattice Boltzmann fluid dynamics.
24  * Header file for \ref lb-boundaries.cpp.
25  *
26  * In the current version only simple bounce back walls are implemented. Thus
27  * after the streaming step, in all wall nodes all populations are bounced
28  * back from where they came from. Ulf Schiller spent a lot of time
29  * working on more powerful alternatives, they are to be found in the
30  * lb_testing branch of espresso until the end of 2010. Now we stripped
31  * down the code to a minimum, as most of it was not sufficiently understandable.
32  *
33  * Anyone who wants to revive these, please look into the git.
34  *
35  */
