Detailed Description

The interface with VMD.

This code just provides a wrapper for the IMD interface functions, which allow to send particle positions to VMD. Additionally, VMD can send back a single integer value, called transfer_rate, which is accessible both from c and from Tcl. The IMD force feedback is not implemented.
